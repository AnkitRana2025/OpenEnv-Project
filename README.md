# OpenEnv Smart Grid Energy Management

## Project Overview
A real-world smart grid environment for training AI agents, built with OpenEnv specification.

## Features
- 🌞 Solar renewable energy integration
- 🔋 Battery storage management
- ⚡ Dynamic grid pricing
- 📊 3 difficulty levels (Easy/Medium/Hard)
- 🎯 Meaningful reward function
- 📈 Baseline agent with reproducible scores
